Adds a floating WhatsApp contact button to your site, configurable from the admin panel.

== Description ==

This plugin adds a floating WhatsApp button to the bottom-right corner of your WordPress site.
You can set the target phone number from the settings page under **Settings > Sticky WhatsApp**.

**Features:**
- Clean and lightweight code
- Mobile and desktop compatibility
- Settings panel to change phone number

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sticky-whatsapp` directory, or install it via the WordPress plugin screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Sticky WhatsApp** to configure the phone number.
